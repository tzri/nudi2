== Theme Name ==

nudi2 – A zero-CSS WordPress theme for the AI-first web.

Author: Mike Moisio
Website: https://mikemoisio.com
Theme: https://mikemoisio.ai/nudi2/

== Description ==

nudi2 is built for a future where content is meant to be parsed, not painted.  
When AI is the primary reader of your website, structure matters more than style.

This theme is almost totally CSS-free. It's fast, accessible, and gives developers a clean slate.

== Features ==

- No-CSS layout – no visual bloat
- AI-friendly HTML structure
- Built-in Schema.org markup (Article, WebPage, Breadcrumbs)
- Clean, accessible, machine-readable
- Style is never out of fashion
